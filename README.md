# Oasis Infobyte Data Science Internship

## 📌 Overview
This repository contains the Data Science projects completed during my Oasis Infobyte Data Science Internship. These projects demonstrate my skills in Python, Machine Learning, Data Analysis, and Data Visualization.

---

## 📂 Projects

### 1. Iris Flower Classification
- Built a Machine Learning model to classify iris flowers into different species.
- Algorithms used: Scikit-learn Classification.
- Technologies: Python, Pandas, NumPy, Matplotlib, Scikit-learn.

### 2. Car Price Prediction
- Developed a regression model to predict car prices based on different features.
- Technologies: Python, Pandas, NumPy, Matplotlib, Scikit-learn.

### 3. Unemployment Analysis with Python
- Analyzed unemployment trends using data visualization techniques.
- Technologies: Python, Pandas, Matplotlib, Seaborn.
